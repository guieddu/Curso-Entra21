Introdução aos Keyframes:

Keyframes são usados em CSS para criar animações suaves e personalizadas em elementos HTML. Eles permitem que você defina diferentes estágios de uma animação e controle as transições entre esses estágios. As principais propriedades relacionadas aos keyframes são:

  • @keyframes
  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-direction

Abaixo podemos ver as explicações de cada uma das propiedades:


@keyframes:

O @keyframes é uma regra especial que define os estágios da animação. Você fornece um nome para o conjunto de keyframes (por exemplo, @keyframes name) e define os estados-chave (porcentagens ou pixels) da animação dentro dele.


@keyframes name { 0% { opacity: 0; } 100% { opacity: 1; } }


animation-name:

Define o nome do conjunto de keyframes a ser usado na animação de um elemento. Exemplo:


animation-name: name;


animation-duration:

Especifica a duração da animação em segundos ou milissegundos. Exemplo:


animation-duration: 2s;


animation-timing-function:

Define a temporização que controla a velocidade e a aceleração de uma animação, podendo ser (ease, linear, ease-in, ease-out, ease-in-out e cubic-bezier()(função que eu posso definir um padrão de minha escolha)) Exemplo:


animation-timing-function: ease-in-out;


animation-delay:

Adiciona um atraso antes do início da animação em segundos ou milissegundos. Exemplo:


animation-delay: 1s;


animation-iteration-count:

Define quantas vezes a animação deve ser reproduzida (ou "infinite" para repetição infinita). Exemplo:


animation-iteration-count: infinite;


animation-direction:

Especifica a direção da animação (normal, reverse, alternate, alternate-reverse). Exemplo:


animation-direction: alternate;
